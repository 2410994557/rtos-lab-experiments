This repository contains RTOS lab experiments performed using STM32F446RE and FreeRTOS in STM32CubeIDE.

1. LED Blinking using GPIO
2. Push Button Interfacing
3. Ultrasonic Sensor Interfacing
4. PWM Generation
5. Super Loop Programming
6. FreeRTOS Basic Task
7. Task Priority Scheduling
8. Binary Semaphore
9. Queue Communication
10. Counting Semaphore
